# SQLCortex VS Code Extension

## Usage
- Run `SQLCortex: Login` and paste your API token.
- Select your org, project, and connection from the SQLCortex activity bar.
- Use `SQLCortex: Run Selection` or `SQLCortex: Analyze Selection` from the editor context menu.
- `EXPLAIN ANALYZE` is opt-in (`sqlcortex.explain.allowAnalyze`) and can execute queries; use it on safe environments only.
